MIME Types API

API to get MIME types of files

Usage: https://mime-types-api.vercel.app/api/ext/{extension}

Example: https://mime-types-api.vercel.app/api/ext/pdf

Endpoints

GET /api/ext/{extension}

Get MIME types of a file extension

Example: https://mime-types-api.vercel.app/api/ext/pdf

Response: 

{
  "data": ["application/pdf"],
  "links": {
    "parent": {
      "href": "https://mime-types-api.vercel.app/api/345"
    }
  }
}

GET /api/{id}

Get a MIME type by its id

Example: https://mime-types-api.vercel.app/api/345

Response: 

{
  "data": {
    "id": 345,
    "name": "application/pdf",
    "description": "Portable Document Format (PDF) is a document file format originally from Adobe, based on PostScript. It has many subsets.",
    "deprecated": null,
    "use_instead": null
  },
  "links": {
    "links": {
      "href": "http://mime-types-api.vercel.app/api/345/links",
      "type": "application/json",
      "method": "GET"
    },
    "extensions": {
      "href": "http://mime-types-api.vercel.app/api/345/extensions",
      "type": "application/json",
      "method": "GET"
    },
    "self": {
      "href": "http://mime-types-api.vercel.app/api/345",
      "type": "application/json",
      "method": "GET"
    }
  }
}

GET /api/{id}/links

Get links of a MIME type by its id

Example: https://mime-types-api.vercel.app/api/345/links

Response: 

{
  "data": {
    "id": 389,
    "mime_type_id": 345,
    "deprecates": "application/x-pdf",
    "related_to": "",
    "parent_of": "",
    "alternative_to": ""
  },
  "links": {
    "parent": {
      "href": "http://mime-types-api.vercel.app/api/345",
      "type": "application/json",
      "method": "GET"
    },
    "self": {
      "href": "http://mime-types-api.vercel.app/api/345/links",
      "type": "application/json",
      "method": "GET"
    }
  }
}

GET /api/{id}/extensions

Get extensions of a MIME type by its id

Example: https://mime-types-api.vercel.app/api/345/extensions

Response: 

{
  "data": [{ "id": 519, "name": ".pdf", "parents": [345] }],
  "links": {
    "parent": {
      "href": "http://mime-types-api.vercel.app/api/345",
      "type": "application/json",
      "method": "GET"
    },
    "self": {
      "href": "http://mime-types-api.vercel.app/api/345/extensions",
      "type": "application/json",
      "method": "GET"
    }
  }
}
